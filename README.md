# LetsKlay

클레이튼을 사용한 크라우드 펀딩 서비스

### 0. 주요 스택
- Node 14
- React + Bootstrap
- NestJS
- Mysql + TypeORM
- Solidity(예정)
  

### 1. 실행 방법
(1) 라이브러리 설치
```bash
$ yarn
```

(2) 서버 실행 
```bash
$ yarn dev #개발 서버

$ yarn storybook #스토리북

$ yarn build # 프로덕션 서버
$ yarn start
```
